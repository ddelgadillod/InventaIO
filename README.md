
# InventAI/o

**Sistema Inteligente de Consulta y Gestión Logística para Distribuidores de Abarrotes**

Intelligent inventory consultation and decision-support system for grocery distributors. PWA with interactive dashboards, ML demand forecasting, automated replenishment recommendations, and a conversational NLP agent. 

---

## Descripción

InventAI/o es un sistema inteligente de consulta y soporte a la toma de decisiones para distribuidores de abarrotes pequeños y medianos. Integra una bodega de datos analítica con esquema estrella, una plataforma web progresiva (PWA) con dashboards interactivos y semáforos de inventario, modelos de machine learning para predicción de demanda y recomendación automatizada de compras y transferencias, y un agente conversacional basado en procesamiento de lenguaje natural que permite consultar el sistema mediante lenguaje cotidiano.

La aplicación no captura datos transaccionales; se especializa exclusivamente en análisis, predicción y recomendación.
